We validated one Amazon Linux 2 EC2 instance running kernel 5.10.251-248.983.amzn2.x86_64. The AWS Amazon Linux Security Center lists CVE-2026-31431 as affecting Amazon Linux 2 kernel and Kernel-5.10 Extra packages with status Pending Fix. The algif_aead module is present and loadable on the instance, but it is not currently loaded and no active AF_ALG usage was observed. Until AWS releases a patched kernel, we should treat the instance as affected and track for remediation.
